# Day 10 – Calculator

## Overview

This project is my solution to the Calculator challenge from Day 10 of Angela Yu's **100 Days of Code: Python Bootcamp**.

The program creates a calculator that performs basic mathematical operations such as addition, subtraction, multiplication, and division.

The calculator allows users to continue calculations using their previous answer or start a new calculation. This project helped me practice functions, return values, dictionaries, and program flow.

## Challenges

The most challenging part of this project was understanding how to control the calculator flow.

At first, I understood how to perform the calculations, but I struggled with allowing the user to continue calculating with the previous answer.

I learned how important program structure is. Understanding where to place loops, how to store the previous result, and how to restart the calculator required more problem-solving.

Another challenge was understanding Angela's method of storing functions inside a dictionary. This helped me see how code can be made cleaner and more efficient.
