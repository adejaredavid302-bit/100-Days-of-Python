## Project Description

This project is a text-based implementation of the classic Hangman game.

The game:
- Randomly selects a secret word from a list of fruits
- Prompts the player to guess one letter at a time
- Reveals correctly guessed letters while hiding the remaining letters
- Tracks previously guessed letters
- Reduces the player's lives for incorrect guesses
- Displays the Hangman drawing after each incorrect guess
- Ends when the player correctly guesses the word or runs out of lives
