# Day 8 – Caesar Cipher

## Overview

This project is my solution to the Caesar Cipher challenge from Day 8 of Angela Yu's 100 Days of Code: Python Bootcamp.

The application allows users to encrypt and decrypt messages by shifting letters through the alphabet using the Caesar Cipher algorithm. I also implemented the ability to restart the program multiple times without rerunning the script.

## What I Learned

- How to create reusable functions with parameters.
- How to import code from another Python file (`art.py`).
- How to use the modulo (`%`) operator to wrap around the alphabet.
- How to encrypt and decrypt text using the Caesar Cipher algorithm.
- How the `continue` statement skips the current iteration of a loop.
- How to restart a program using a `while` loop.
- How to use `list.index()` to locate a letter inside a list.
- How Python handles negative modulo values, such as `-2 % 26 == 24`.
